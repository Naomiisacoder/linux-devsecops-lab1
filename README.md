# Linux & DevSecOps Access Control Simulation

Author: Naomi Onwuka  
Role Simulation: DevSecOps Engineer  
Organization Context: ParoCyber LLC Assignment

This repository documents the execution of a Linux security and DevSecOps access control simulation.

The objective was to demonstrate:
- Password state investigation
- Secure remediation procedures
- Department-based access control
- Service account configuration
- Privilege escalation control
- Audit-aware user lifecycle management

  ## Skills Demonstrated
- Linux user management
- /etc/shadow password inspection
- Group management and access control
- Sudo privilege configuration
- Service account security hardening
- Audit-conscious user deletion
- Terminal-based documentation

  ## Evidence Structure
  Each scenario contains:
- Commands executed
- Terminal output
- Screenshots
- Security observations
- System implications
  
  ## Repository Structure

Each scenario contains:

- `commands.md` → commands executed
- `explanation.md` → observations and breakdown
- `screenshots/` → terminal screenshots showing proof
